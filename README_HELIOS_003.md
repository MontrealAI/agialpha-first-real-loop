# AGI ALPHA HELIOS-003 v0.1

**Public Benchmark Bridge and Delayed-Outcome Gauntlet for Governed Compounding of Verified Machine Labor**

HELIOS-003 is a bounded, autonomous, local/proxy Evidence Docket experiment. It tests whether reusable capability from the HELIOS lineage transfers to harder cross-domain tasks while remaining replayable, baseline-comparative, safety-ledgered, cost-ledgered, externally reviewable, and claim-bounded.

## What it demonstrates

- L4-ready external reviewer kit.
- L5-local transfer baseline comparison with B0-B6 ladder.
- L6-CI scaling proxy across agent/node counts.
- L7-local cross-domain task portfolio.
- L8 delayed-outcome sentinel readiness.
- External benchmark adapter templates for SWE-bench-style, GAIA-style, tau-bench-style, BrowserGym/OSWorld-style, and AGI Jobs protocol-native tasks.

## What it does not claim

It does not claim achieved AGI, ASI, empirical SOTA, safe autonomy, standard-setting control, guaranteed economic return, real-world energy savings, or civilization-scale capability.

## Main command

```bash
python -m agialpha_helios3 run --out runs/helios-003/local
python -m agialpha_helios3 replay --root runs/helios-003/local
python -m agialpha_helios3 falsification --root runs/helios-003/local
```
