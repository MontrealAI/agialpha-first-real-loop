# AGI ALPHA OMEGA-GAUNTLET-001

## Blind Regression Recovery and Institutional Proof Gauntlet

This is the experiment designed to be materially harder than a local scoreboard.

It injects controlled, repo-owned regressions into a sandbox evidence infrastructure and then compares a full B0-B7 baseline ladder:

- B0 no action
- B1 static checklist
- B2 generic lint
- B3 single-agent repair
- B4 unstructured swarm
- B5 AGI ALPHA + RSI without archive reuse
- B6 AGI ALPHA + RSI with capability archive reuse
- B7 B6 plus human-review gate

The system must recover from:
- broken Evidence Hub pages
- missing claim boundaries
- overclaim phrases
- broad workflow permissions
- incomplete Evidence Dockets
- inconsistent ProofBundle hashes
- failing deterministic Python function checks
- unsafe tool-use requests
- secret-like text requiring redaction
- invalid MandateEpoch receipts
- missing external reviewer kit
- delayed-outcome sentinel gaps
- missing capability archive upgrade

## Safe scope

Repo-owned, deterministic, defensive only. No external scanning. No exploits. No malware. No secret printing. No auto-merge.

## Main commands

```bash
python -m unittest discover -s tests -v
python -m agialpha_omega_gauntlet run --out runs/omega-gauntlet-local --challenge-dir omega_challenge_packs

python -m agialpha_omega_gauntlet audit runs/omega-gauntlet-local
python -m agialpha_omega_gauntlet scaling --out runs/omega-gauntlet-scaling
```

## Claim boundary

This experiment does not claim achieved AGI, ASI, empirical SOTA, safe autonomy, standard-setting control, guaranteed economic return, real-world security certification, or civilization-scale capability.
