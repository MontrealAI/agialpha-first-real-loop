## Run AGI ALPHA Experiments from GitHub Web UI
1. Open Actions.
2. Choose workflow.
3. Run workflow.
4. Wait for green success.

Green checkmark means workflow completed; it does not mean empirical SOTA.
Artifact uploaded means evidence exists; it does not mean external review happened.
L4-ready means replay kit exists. L4-external means an external reviewer attested.

No Evidence Docket, no empirical SOTA claim. Autonomous evidence production is allowed; autonomous claim promotion is not.
