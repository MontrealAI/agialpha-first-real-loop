# Evidence Docket Standard

A completed loop must produce:

1. Seed ID.
2. MARK review card.
3. Sovereign ID.
4. Job list.
5. Agent or workflow outputs.
6. Sources used.
7. Reviewer decisions.
8. Red-team notes.
9. Final accepted interventions.
10. Reusable constructs extracted.
11. Lessons for next seed.
12. Treatment/control comparison.

## Rule

If it is not in the Evidence Docket, it did not happen.

## External proof upgrade

For publication-grade proof, add:

- archived source snapshots;
- independent reviewer identity;
- baseline runs;
- cost ledger;
- safety ledger;
- replay logs;
- failure cases;
- repeated-trial reliability estimate.
