# Repository Map

Top-level operator map of code, workflows, evidence artifacts, and docs.

- `.github/workflows/`: all automation workflows.
- `docs/`: operator/reviewer/contributor docs and SecureRails policy boundary docs.
- `scripts/`: claim/safety/use-case/no-automerge checks plus docs tooling.
- `tests/`: unit and documentation verification tests.
- `evidence_registry/`, `cyber-ga-sovereign-runs/`, `agiga-foundry-runs/`, `rsi_state/`: generated evidence and run state.
- `agialpha_*` packages: experiment family implementations.

Status terms used in docs: planned, pending, scaffolded, implemented, evidence-backed, external-review pending.
