# Claim Boundary

This repository does not claim AGI, ASI, empirical SOTA, energy abundance, facility-specific engineering authority, guaranteed savings, or autonomous deployment authority.

It claims only a replayable first-loop scaffold:

- one Nova-Seed;
- one MARK review;
- one mini Sovereign;
- five jobs;
- one Evidence Docket;
- one reusable compiler;
- one vNext treatment/control comparison.

External proof requires independent review, archived source verification, baseline comparison, safety/cost ledgers, and repeated runs.
