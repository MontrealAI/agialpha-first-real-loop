# Run a workflow

- Evidence Mission Control publisher: run `evidence-hub-publish` workflow.
- SecureRails guard: run `secure-rails-compliance-guard` workflow or scripts in `scripts/`.
- Replay: run `replay.yml` or family-specific replay workflows.
- Docs audit: run `docs-audit.yml` or `python -m agialpha_docs ...`.
- AGI-GA / Cyber-GA / RSI: run corresponding `*-autonomous.yml` workflows in Actions.
