# One-page summary

The first real AGI ALPHA loop is deliberately small. It tests whether a machine-discovered opportunity can be represented as a Nova-Seed, reviewed by MARK, instantiated as a mini Sovereign, executed through five jobs, proven through an Evidence Docket, converted into a reusable compiler, and used to improve a second seed.

The first domain is cold-chain energy efficiency because it is useful, energy-relevant, public-evidence-friendly, commercially meaningful, and safe if kept advisory.

The first proof is not that agents completed a task. The first proof is that completed, reviewed, evidence-bearing work improves the next seed.
