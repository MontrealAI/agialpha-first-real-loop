## Objective

Prove that AGI ALPHA can turn one validated sovereign opportunity into reusable leverage for the next cycle.

## Loop

```text
Nova-Seed → MARK → Mini Sovereign → 5 AGI Jobs → Evidence Docket → Compiler-v0 → Seed-vNext → treatment/control
```

## Domain

Cold-chain energy efficiency, restricted to advisory low-capex electricity-saving recommendations for refrigerated warehouses and food-processing cold-chain facilities.

## Run command

```bash
python -m agialpha_first_loop.run --out runs/coldchain-energy-loop-001
```

## Output

The Evidence Docket is written to:

```text
runs/coldchain-energy-loop-001/
```

## Pass/fail rule

The loop passes only if Seed-002 with `ColdChain-Energy-Compiler-v0` improves accepted-output yield by at least 25% over Seed-002 without the compiler, with no increase in hallucinated sources or safety issues.
