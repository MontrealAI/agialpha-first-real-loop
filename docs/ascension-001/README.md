# ASCENSION-001 — Governed Recursive Self-Improvement of the Evidence Institution

**Purpose:** prove, in a bounded CI/proxy way, that AGI ALPHA can improve the institution that evaluates AGI ALPHA.

The experiment runs a deterministic RSI loop:

```text
repo scan → RSI state hash → B0-B7 baselines → proof bundles → replay → ECI ledger
→ capability archive update → held-out transfer → Move-37 dossier gate → external replay kit
```

It tests the central compounding question:

```text
Does EvidenceCompilerCapabilityArchive-vN improve the next evidence task
more than the same system with no archive reuse?
```

Claim boundary: This experiment does not claim achieved AGI, ASI, empirical SOTA, safe autonomy, real-world certification, guaranteed economic return, or civilization-scale capability. It is a bounded, repo-owned, CI/proxy Evidence Docket experiment testing whether governed RSI state, replay, baselines, and capability archives can improve future verified work.

No Evidence Docket, no empirical SOTA claim. Autonomous evidence production is allowed; autonomous claim promotion is not.
