# Adding New Workflows

Rules:
- Every workflow must be documented.
- No direct Pages deployment unless central publisher.
- Use `workflow_dispatch` inputs with safe defaults.
- Apply least-privilege permissions.
- Upload artifacts and evidence manifests where applicable.
- Include safety and claim boundary checks.

## Checklist
- [ ] Workflow file exists
- [ ] Workflow catalog updated
- [ ] README link added if needed
- [ ] Evidence manifest emitted if experiment workflow
- [ ] Artifacts uploaded
- [ ] Safety boundary included
- [ ] No direct Pages deploy unless central publisher
- [ ] Tests pass
