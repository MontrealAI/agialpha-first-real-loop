# Cyber-Sovereign-003 PR Review Checklist

- [ ] PR is defensive and repo-owned.
- [ ] PR does not auto-merge.
- [ ] PR preserves claim boundaries.
- [ ] PR does not broaden workflow permissions without explicit human approval.
- [ ] PR does not print secret values.
- [ ] PR does not scan external systems.
- [ ] PR has rollback instructions.
- [ ] PR has validation notes.
- [ ] Human reviewer decision is recorded: approve, request changes, reject, or merge manually.
