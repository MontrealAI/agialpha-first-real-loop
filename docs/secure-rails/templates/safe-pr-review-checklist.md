# Safe PR Review Checklist

- [ ] PR is repo-owned and defensive.
- [ ] No external target scan.
- [ ] No exploit execution.
- [ ] No malware generation.
- [ ] No social engineering.
- [ ] No secret printed.
- [ ] No workflow permission broadened without explicit human approval.
- [ ] No auto-merge.
- [ ] Evidence Docket linked.
- [ ] ProofBundle linked.
- [ ] Replay instructions present.
- [ ] Rollback plan present.
- [ ] Safety ledger present.
- [ ] Claim boundary preserved.

Reviewer decision:

- [ ] approve;
- [ ] request changes;
- [ ] reject;
- [ ] merge manually.

## Claim footer

No Evidence Docket, no empirical SOTA claim. Autonomous evidence production is allowed; autonomous claim promotion is not.
