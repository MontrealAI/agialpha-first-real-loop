# SecureRails Claims and Marketing Guardrails

## Approved tagline

SecureRails makes AI-agent work safe to review, safe to replay, and safe to remediate.

## Approved one-liner

SecureRails is AI-agent security governance and proof-bound defensive remediation for autonomous software work.

## Approved longer paragraph

SecureRails converts agent actions, workflow changes, findings, and remediation proposals into replayable ProofBundles, Evidence Dockets, redacted safety ledgers, safe PRs, and reusable defensive capability — with human-governed promotion and no autonomous claim escalation.

## Forbidden claims

Do not claim:

- achieved AGI;
- achieved ASI;
- empirical SOTA;
- safe autonomy;
- official benchmark victory;
- real-world security certification;
- guaranteed security;
- guaranteed compliance;
- guaranteed economic return;
- investment opportunity;
- offensive cyber capability;
- autonomous production remediation.

## Replacement language

| Unsafe phrase | Use instead |
|---|---|
| certified secure | evidence-backed defensive review |
| autonomous security fix | human-reviewed safe remediation proposal |
| guaranteed compliance | compliance-supporting evidence workflow |
| exploit agent | defensive validation workflow |
| investment return | protocol utility operation |
| yield | validator fee / protocol fee under stated assumptions |

## Claim footer

No Evidence Docket, no empirical SOTA claim. Autonomous evidence production is allowed; autonomous claim promotion is not.
