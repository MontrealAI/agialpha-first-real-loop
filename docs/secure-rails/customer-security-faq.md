# Customer Security FAQ

- **What is SecureRails?** AI-agent security governance and proof-bound defensive remediation.
- **Is SecureRails a certification?** No. It provides readiness evidence and advisory artifacts, not certification.
- **Does SecureRails execute code?** It can run controlled validations and checks with human review gates.
- **Does SecureRails scan external targets?** No.
- **Does SecureRails print secrets?** No; redaction and non-printing boundaries apply.
- **Does SecureRails auto-merge?** No.
- **Does SecureRails make decisions about people?** No by intended purpose.
- **Is it GPAI provider by default?** No.
- **EU AI Act high-risk?** SecureRails is not intended, designed, validated, or authorized for Annex III high-risk AI use cases.
- **$AGIALPHA usage?** Utility-only boundary; not financial-product rights.
- **Artifacts?** ProofBundles, Evidence Dockets, Work Vault records, and advisory logs.
