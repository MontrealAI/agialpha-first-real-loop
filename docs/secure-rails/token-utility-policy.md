# $AGIALPHA Token Utility Policy

## Default position

`$AGIALPHA` is utility infrastructure for protocol operations. It is not equity, debt, yield, dividends, ownership, profit rights, entity ownership, output rights, guaranteed appreciation, guaranteed operating surplus, or a financial product.

## Permitted utility functions

- Protocol access.
- Staking for Sybil resistance.
- Escrow for AGI Jobs.
- Settlement for validated work.
- Validator fees.
- Slashing for validator or agent misconduct.
- α-Work Unit accounting.
- ProofBundle registration.
- Evidence Docket anchoring.
- Capacity allocation under stated assumptions.
- Protocol operations.

## Forbidden language

Do not say:

- investment;
- yield;
- dividend;
- profit share;
- ownership share;
- equity;
- debt;
- guaranteed return;
- guaranteed appreciation;
- passive income;
- asset-backed return;
- revenue rights;
- security token, unless counsel explicitly approves a regulated offering.

## Public token sentence

Use this:

> `$AGIALPHA` is utility infrastructure for protocol operations: staking, Sybil resistance, escrow, settlement, validator fees, slashing, α-WU accounting, ProofBundle registration, Evidence Docket anchoring, and capacity allocation under stated assumptions. It is not marketed as an investment product, ownership interest, yield instrument, dividend right, or guaranteed appreciation mechanism.

## Token launch gate

Before any public sale, exchange listing, market-making arrangement, staking reward communication, validator reward communication, or treasury discussion:

- [ ] Token utility review completed.
- [ ] MiCA / securities / financial promotions counsel review completed.
- [ ] No investment language in public pages.
- [ ] No guaranteed return language.
- [ ] No yield or profit-right language.
- [ ] Token use tied to actual protocol operation.

## Claim footer

No Evidence Docket, no empirical SOTA claim. Autonomous evidence production is allowed; autonomous claim promotion is not.
