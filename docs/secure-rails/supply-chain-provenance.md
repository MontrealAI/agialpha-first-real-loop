# SecureRails Supply-Chain Provenance 001

SecureRails adds supply-chain evidence for Work Vaults, ProofBundles, Evidence Dockets, and registry artifacts.

It produces artifact hash manifests, local provenance records, optional GitHub Artifact Attestation records, and repository security health reports.

GitHub Artifact Attestations can support SLSA-style provenance. This repository does not claim a SLSA level unless explicitly verified and documented.

No Evidence Docket, no empirical SOTA claim. Autonomous evidence production is allowed; autonomous claim promotion is not.
