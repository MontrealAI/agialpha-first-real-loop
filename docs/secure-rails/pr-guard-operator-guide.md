# PR Guard Operator Guide

SecureRails Agentic PR Guard 001 is an advisory governance control. It does not certify security and does not auto-merge.

> No Evidence Docket, no empirical SOTA claim. Autonomous evidence production is allowed; autonomous claim promotion is not.

1. Open PR
2. Wait for SecureRails Agentic PR Guard 001
3. Open check summary
4. Download artifact
5. Read `summary.md`
6. Inspect the Evidence Docket (`11_evidence_docket/`)
7. Check `09_safety_ledger.json` and `12_validator_report.json`
8. Make a human review decision: safe remediation, reject, or escalate
9. Do not treat SecureRails output as certification
10. Merge only through normal human process if appropriate
