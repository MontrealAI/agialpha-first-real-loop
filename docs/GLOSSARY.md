# Glossary

- **AGI ALPHA**: bounded research program and governance stack.
- **Evidence Mission Control**: evidence publishing and navigation layer.
- **SecureRails**: AI-agent security governance and proof-bound defensive remediation.
- **Work Vault**: governed record of scoped defensive work.
- **ALPHA AGI MARK**: utility budget allocation marker.
- **ALPHA AGI Sovereign**: governed execution identity assignment.
- **AGI Job**: bounded unit of agent work.
- **ProofBundle**: artifact bundle with integrity metadata.
- **Evidence Docket**: structured evidence package for conservative claims.
- **safety ledger**: run safety counters and guardrail events.
- **cost ledger**: utility/resource accounting record.
- **claim boundary**: explicit wording limits for allowed vs forbidden claims.
- **validator**: checker that scores and gates outcomes.
- **replay**: rerun process to test reproducibility.
- **falsification audit**: attempt to disconfirm claimed behavior.
- **safe PR**: proposed change requiring human review before merge.
- **CyberSecurityCapabilityArchive / QDArchive / vNext**: defensive capability and evolution records.
- **B0-B7, B5/B6/B7**: internal baseline maturity ladder.
- **$AGIALPHA**: protocol utility accounting rail, not investment framing.
- **α-Work Unit**: normalized unit of validated work accounting.
- **utility settlement**: protocol settlement record for validated utility activity.
- **human-governed promotion**: claims/actions approved by humans.
- **autonomous claim promotion**: forbidden automatic claim escalation.
- **local/external/official benchmark evidence**: evidence provenance tiers.

- **RSI Governor**: governance subsystem for bounded recursive self-improvement experiments with replay and falsification gates.
