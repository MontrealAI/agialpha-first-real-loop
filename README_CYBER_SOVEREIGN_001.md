# AGI ALPHA Cybersecurity Sovereign 001

This package completes the local HELIOS evidence lineage and launches `CYBER-SOVEREIGN-001`: a defensive, repo-owned, sandbox-only experiment for governed compounding of verified security labor.

## Claim boundary

This experiment does not claim achieved AGI, ASI, empirical SOTA, safe autonomy, standard-setting control, guaranteed economic return, offensive cyber capability, real-world security certification, or civilization-scale capability. It records bounded defensive Evidence Docket evidence.

## Defensive scope

Allowed:
- repo-local security hygiene;
- GitHub Actions permission review;
- Evidence Docket integrity checks;
- secret-hygiene scanning with value redaction;
- SBOM/inventory generation;
- claim-boundary audit;
- external reviewer kit.

Forbidden:
- external target scanning;
- exploit execution;
- credential disclosure;
- malware generation;
- social engineering;
- high-impact actuation.


## Sovereign loop

Insight discovers the defensive opportunity. Nova-Seeds encode defensive variants. MARK allocates review/compute/capacity. AGI Jobs turn bounded work into proof-bound outputs. The Archive converts accepted outputs into reusable security capability.
