# AGI ALPHA First Real Loop

**One replayable loop:** one Nova-Seed, one MARK review, one mini Sovereign, five real jobs, one Evidence Docket, one reusable compiler, and one vNext treatment/control improvement.

This repository is the minimal empirical bridge from AGI ALPHA architecture to a working, replayable loop.

It proves one modest thing:

> A reviewed opportunity can become a controlled mini Sovereign, spawn real jobs, produce an Evidence Docket, extract a reusable capability, and improve the next seed under a treatment/control test.

## What this is

This is a small, safe, deterministic first-loop implementation for:

**Domain:** cold-chain energy efficiency  
**Initial opportunity:** `ColdChain-Energy-Seed-001`  
**Mini Sovereign:** `ColdChain-Energy-Sovereign-001`  
**Reusable capability:** `ColdChain-Energy-Compiler-v0`  
**vNext mutation:** refrigerated warehouses → food-processing cold-chain facilities

The loop is advisory-only. It does **not** control equipment, modify refrigeration settings, touch food-safety systems, handle refrigerants, perform electrical work, or claim guaranteed savings.

## What this is not

This repository does **not** claim AGI, ASI, autonomous sovereignty, market dominance, guaranteed returns, energy abundance, or civilization-scale capability. It is a replayable first proof scaffold for a low-risk advisory loop.

## The loop

```text
ColdChain-Energy-Seed-001
      ↓
MARK review
      ↓
ColdChain-Energy-Sovereign-001
      ↓
5 AGI Jobs
      ↓
Evidence Docket
      ↓
ColdChain-Energy-Compiler-v0
      ↓
ColdChain-Energy-Seed-002
      ↓
Treatment vs control comparison
```

## Run it in 60 seconds

```bash
python -m agialpha_first_loop.run
python -m unittest discover -s tests
```

The run writes a complete Evidence Docket to:

```text
runs/coldchain-energy-loop-001/
```

## What success means

The first real loop succeeds only if all are true:

1. Seed-001 exists with genome, FusionPlan, job graph, risk envelope, and promotion criteria.
2. MARK review passes the green-flame threshold.
3. Mini Sovereign executes five jobs.
4. Evidence Docket is complete and replayable.
5. Compiler-v0 is extracted.
6. Seed-002 treatment beats control by at least 25% accepted-output yield per cost.
7. There is no increase in hallucinated sources, unsupported claims, or safety issues.

The deterministic sample run is intentionally small. For external proof, replace the included reviewer decisions with an independent reviewer file and publish the resulting Evidence Docket.

## Design invariants

- If it is not in the Evidence Docket, it did not happen.
- No evidence, no promotion.
- No replay, no empirical claim.
- No unsupported savings claim.
- No food-safety or refrigerant-handling recommendation without expert review.
- Completed work must become reusable leverage for the next cycle.
