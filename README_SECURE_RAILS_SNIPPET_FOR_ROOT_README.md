## AGI Alpha SecureRails

**SecureRails is AI-agent security governance and proof-bound defensive remediation.** It secures the rails for autonomous software work by converting agent actions, workflow changes, findings, and remediation proposals into replayable ProofBundles, Evidence Dockets, redacted safety ledgers, safe PRs, and reusable defensive capability, with human-governed promotion and no autonomous claim escalation.

SecureRails makes AI-agent work safe to review, safe to replay, and safe to remediate.

Boundary:

SecureRails is defensive AI-agent governance and proof-bound remediation. It is not autonomous cybersecurity certification, not offensive cyber, not a high-risk decision system by intended purpose, not a GPAI model provider by default, and not an investment product.

No Evidence Docket, no empirical SOTA claim. Autonomous evidence production is allowed; autonomous claim promotion is not.
