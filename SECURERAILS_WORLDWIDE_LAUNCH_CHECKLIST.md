# SecureRails Worldwide Launch Checklist

## Repository upload

- [ ] Package uploaded on a new branch.
- [ ] Existing root README preserved.
- [ ] SecureRails README snippet added to root README.
- [ ] Compliance Guard workflow added.
- [ ] Pull Request opened.
- [ ] Pull Request reviewed manually.

## EU AI Act hardening

- [ ] Intended purpose statement is present.
- [ ] Foreseeable misuse and excluded uses page is present.
- [ ] Customer use attestation template is present.
- [ ] Annex III triage template is present.
- [ ] Material modification and re-screening log is present.
- [ ] No HR / no worker evaluation policy is present.
- [ ] No critical-infrastructure safety-component policy is present.
- [ ] GPAI dependency boundary is present.
- [ ] Regulator audit readiness pack is present.

## Operational controls

- [ ] Human review required before remediation persistence.
- [ ] Auto-merge disabled.
- [ ] No external target scanning.
- [ ] No exploit execution.
- [ ] No malware generation.
- [ ] No social engineering.
- [ ] No raw secret printing.
- [ ] Safety ledger example passes.
- [ ] Claim-boundary guard passes.
- [ ] No-automerge guard passes.
- [ ] Use-case triage guard passes on the example intake.

## Marketing and commercial controls

- [ ] Public copy says SecureRails is AI-agent security governance and proof-bound defensive remediation.
- [ ] Public copy does not claim cybersecurity certification.
- [ ] Public copy does not claim empirical SOTA.
- [ ] Public copy does not claim achieved AGI or ASI.
- [ ] Public copy does not claim safe autonomy.
- [ ] Public copy does not claim guaranteed security or guaranteed compliance.
- [ ] `$AGIALPHA` language is utility-only.

## Customer deployment controls

- [ ] Customer completed deployment intake.
- [ ] Customer signed use attestation.
- [ ] Customer prohibited HR, profiling, critical-infrastructure safety reliance, and automated decisioning uses.
- [ ] Customer confirmed human review for all remediation.
- [ ] Customer confirmed no auto-merge.
- [ ] Deployment-specific data-protection review completed.
- [ ] Regulated-sector deployment escalated to counsel if applicable.

## Launch gate

Launch is permitted only when all above checks are complete or explicitly waived in writing by accountable leadership and counsel.
